What to expect

This documentation is not a tutorial. It will not teach you HTML, CSS or Javascript.

The documentation is about giving you an overview of the structure of that theme. After reading the documentation you know how to use and customize elements of the theme.

Frequently asked questions

What scripts do I need to include to get the plugin plugin-name working?

Either you look on the includes on the sample page or you go through the css/js folders and include the files which are in the plugin-name/ folder. Feel free to contact me if you need help.

The element does not look like in your template

You probably forgot a class-name or used a wrong structure. I would suggest you to inspect the template with FireBug or Chrome developer tools.

What are the mandatory plugins which need to be included in every file?

Please see the blank page (more-blank.html) to see the mandatory includes. Every include except the demonstration.js is needed in every file!
